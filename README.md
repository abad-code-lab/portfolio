# Portfolio - Abdoulaye DAFFE 🚀

Bienvenue sur le dépôt de mon portfolio personnel ! 

Ce site web interactif a été conçu pour présenter mon parcours, mes compétences, mes projets ainsi que mes certifications en tant que développeur fullstack. Il a été développé avec une attention particulière portée au design moderne, à la fluidité et aux performances.

## 🌟 Fonctionnalités Principales

- **Page d'Accueil dynamique :** Présentation globale, aperçu des stacks techniques utilisés, derniers projets en vedette et mes certifications (Nmap / Coursera).
- **À propos :** Détails sur ma formation (Master 2 en Génie Logiciel), mon parcours et mes centres d'intérêt (IA, Cloud, Cybersécurité, Big Data).
- **Compétences :** Liste exhaustive des technologies maîtrisées tant sur le frontend (React, Angular) que le backend (Spring Boot, Java, Python).
- **Projets :** Présentation détaillée de mes réalisations (Gestion de notes étudiantes, Plateforme pour ouvriers, etc.).
- **Responsive Design :** L'interface s'adapte parfaitement à tous les types d'écrans (Mobile, Tablette, Desktop).
- **Téléchargement direct du CV :** Possibilité de télécharger facilement mon curriculum vitae en version PDF depuis le site.

## 🛠️ Technologies Utilisées

Ce projet a été construit en utilisant les technologies modernes suivantes :

### Frontend
- **[React](https://reactjs.org/) (v18)** - Bibliothèque JavaScript pour construire des interfaces utilisateurs interactives.
- **[Vite](https://vitejs.dev/)** - Outil de build frontend rapide de nouvelle génération.
- **[Tailwind CSS](https://tailwindcss.com/) (v3)** - Framework CSS utilitaire pour un design rapide et sur-mesure.
- **[React Router](https://reactrouter.com/) (v6)** - Pour la navigation fluide entre les différentes pages de l'application sans rechargement.
- **[Lucide React](https://lucide.dev/)** - Pour l'intégration d'icônes élégantes et modernes.

### Services & Outils
- **[EmailJS](https://www.emailjs.com/)** - Service utilisé pour envoyer directement des e-mails depuis le formulaire de contact (géré côté client).
- **ESLint & PostCSS** - Pour garantir la qualité du code et le traitement CSS automatisé.

## 🚀 Installation et Lancement Local

Pour cloner et exécuter cette application sur votre machine locale, vous aurez besoin de [Node.js](https://nodejs.org/en/) installé.

1. **Cloner le répertoire :**
   ```bash
   git clone https://github.com/abad-code-lab/portfolio_abad.git
   cd portfolio_abad
   ```
   *(Note: Remplacez l'URL par la bonne si le dépôt change de nom)*

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement :**
   ```bash
   npm run dev
   ```

4. **Accéder à l'application :**
   Ouvrez votre navigateur web et rendez-vous sur : `http://localhost:5173/`
